# Generated output artefacts

This folder is generated by the GitHub Action.

Publication-level exports are written as partitioned, zstd-compressed Parquet
directories, for example:

```text
data/outputs/scraped_publications_ndph_parquet/publication_year=2024/part-00000.parquet
```

CSV exports are intentionally not generated or committed because they can exceed
GitHub's single-file limit and are much larger than Parquet.
